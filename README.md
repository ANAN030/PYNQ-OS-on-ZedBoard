# PYNQ OS on ZedBoard

用兩支影片分別演示如何將PYNQ OS移植到ZedBoard開發板上，分別是演示如何編譯ZedBoard專用的啟動檔，還有如何把啟動檔移植到PYNQ Image燒錄後的SD卡中。

假如不想自己編譯，可以直接下載我編譯完成的啟動檔，直接從第二部影片開始。
